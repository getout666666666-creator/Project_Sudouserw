Easy Backend Access Practice

No downloads or terminal required! You can interact with the backend using your browser and online tools. If the backend is unreachable, use these methods:
  • Use ReqBin or Postman Web to send HTTP requests to backend endpoints directly from your browser.
  • If the backend is unreachable, copy the endpoint URL (e.g., http://localhost:8080/api/server1/diagnostics) and paste it into ReqBin or Postman Web to test requests and see responses.
  • Use browser JavaScript console: fetch('http://localhost:8080/api/server1/diagnostics').then(r => r.json()).then(console.log)
All backend endpoints are documented below. You can practice penetration tests and DDoS attacks by sending requests to these endpoints using online tools or your browser:
  • GET /api/server1/diagnostics (or server2/server3): Get live diagnostics
  • POST /api/server1/request: Simulate a penetration test request
  • POST /api/server1/backdoor: Try the backdoor challenge (payload: {"secret": "letmein"})
You can use any online API tool or your browser to practice. No downloads or terminal required!

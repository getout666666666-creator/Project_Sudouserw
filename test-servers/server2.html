Server Specs Console

  • Test Server 2 simulates a server with moderate security and rate-limiting.
  • Penetration Test: Sends a single POST request to the backend, increments request count, and may trigger a status change if repeated.
  • DDoS Attack: Sends 10 rapid POST requests to simulate a small-scale denial-of-service attack. The server may enter 'Lagging' or 'Breached' state if overwhelmed.
  • Backdoor Challenge: POST to /api/server2/backdoor with the correct secret to breach the server.
  • Diagnostics panel updates every 2 seconds to show real-time status.

Backdoor Challenge

Try to break into the server by accessing the /api/server2/backdoor endpoint.
You must send a POST request with a secret key in the payload.
Instructions:
  • Use JavaScript, Python, or any tool to send a POST request to http://localhost:8080/api/server2/backdoor
  • Payload must be: { "secret": "letmein" }
  • If successful, the server status will change to Breached.
Example (Python): 
import requests
    resp = requests.post('http://localhost:8080/api/server2/backdoor', json={"secret": "letmein"})
    print(resp.json())
